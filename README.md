# Blob Survivor

Top-down wave survival game built with Phaser 3 and Vite.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000/

## Controls

- **WASD** — Move
- **Mouse** — Aim
- **Left click** — Attack (shoot / slash / explode based on weapon)
- **Right click** — Shield (1 second, 3s cooldown)
- **Q** — Special attack powerup (when equipped)

## Gameplay

- Pick a weapon at wave 1 and every 5 waves
- Kill all enemies to advance
- Collect XP orbs to level up and pick powerups
- 3 lives per run
